TOPIC

This website acts as a portfolio that catalogues a lot of my experiences in a fun and interactive way!

PURPOSE AND DESCRIPTIONS

HOME

  • This page gives a small insight into me (and my personality)
  • The intro animation uses a Lottie file of my initials
  • The first div uses animations to scroll hello (in a plethora of languages)
  • The second div uses transformations to scale a photo of me, which acts as the top of the "i" in CHRIS
  • I summarize my motivations and skills in the second two divs, using flexboxes and "typed" plugins.
  • The final div uses sprite images to reveal lottery balls which navigate the user to other pages.

WORK

  • This page summarizes much of my work experiences
  • I used ::after psuedo-elements to add timelines and numbering of the experiences
  • Each experience is animated upon hovering to emphasize whichever experience the user is lookign at

DESIGNS

  • This page shows some of the designs I've made on SolidWorks
  • I used GIFs to display animated 3D revolutions of my CAD files
  • All of the lines are animated to emphasize upon hovering

CONTACT

  • This page contains several ways to contact me, including my resume
  • Each of the contact elements scale upon hovering, and are attached to a Lottie animation
  • The map was generated using Mapbox, which required generating an API
  • The SOCIALS contains my Instagram and LinkedIn
  • All links open in a separate tab

EXTRAS

Transitions & Transmormations: Used in transformation of "HELLO" scroll, my face, "a little about me", the skills section, the navbar, and most elements of the WORK and DESIGNS section

Used changes in width to reveal wipe animations in WORK and DESIGNS, change in opacity upon hovering, translation of my picture, and translation of 3 of the same text elements for "ABOUT ME" and "SKILLS" in the home page.

JS or jQuery: Used in order to animate the balls upon hover by translating sequentially through the sprite image upon hover;

Further, used in order to prevent the user from scrolling when the intro animation plays, then revealing the navbar and hiding the intro animation div.
